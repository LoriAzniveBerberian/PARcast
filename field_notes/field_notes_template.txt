HOW THE INSTRUMENTS WORK
Both units run automatically when powered. No laptop is needed in the field.

Start a deployment:
    1. Plug power bank into the Teensy
    2. Wait 3 seconds for startup
    3. Remove the PAR sensor cap

Stop a deployment:
    1. Replace the PAR sensor cap
    2. Unplug the power bank

Start a new deployment the same day:
    Just unplug and replug; a new file is created automatically
    (e.g., 0001, 0002).

Reset button (small button on the Teensy):
    DO NOT press during a deployment. It interrupts logging and creates a new file. If pressed accidentally, note the time below.

TROUBLESHOOTING
Timestamps show 2018:
    RTC coin cell (CR1220) lost power. Reseat (+ side up) and
    re-upload rtc_test.ino to set the time. Discard 2018 data.

Multiple files created unexpectedly:
    Reset button was pressed accidentally. Note affected files
    and times — the data is still usable, just split.

SD card not detected:
    Push card in firmly until it clicks. Verify FAT32 + MBR
    format (see SD_CARD_FORMAT_INSTRUCTIONS.txt).
